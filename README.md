# 🚀 CI/CD For Real

Projeto criado para aprender e praticar o uso do GitHub Actions aplicando conceitos de **CI/CD (Continuous Integration e Continuous Delivery)**.

## 📌 Objetivo

Este repositório tem como foco:

- Entender na prática como funciona um pipeline de CI/CD
- Automatizar processos de build, testes e deploy
- Utilizar o GitHub Actions para integração contínua
- Simular um fluxo real de desenvolvimento

## ⚙️ Tecnologias utilizadas

- GitHub Actions
- Node.js / Yarn
- (adicione outras tecnologias do projeto aqui)

## 🔄 Pipeline CI/CD

O fluxo automatizado inclui:

1. Instalação de dependências
2. Execução de testes
3. Verificação de erros
4. (Opcional) Deploy automático
